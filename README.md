## Setup local environment
### Clone
```
git clone https://github.com/mrakibmiah/scrumpoker.git
```

go to folder

```
cd scrumpoker
```

### Install

```
npm i
```

or

```
yarn
```

### Spin up the server

```
npm run dev
```

open [localhost](http://localhost:3005/)
